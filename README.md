# 🍔 Online Food Delivery Time & Rating Analysis

## 📌 Project Overview
This project analyzes online food delivery data using Microsoft Excel to understand delivery performance, customer satisfaction, and revenue patterns.

The dataset contains 20 food delivery orders across multiple Indian cities and food categories.

---

## 🎯 Aim
To analyze food delivery data using Excel functions and generate meaningful insights related to delivery time, ratings, and revenue.

---

## 📊 Objectives
- Apply Excel functions like SUM, AVERAGE, COUNT, IF, RANK, etc.
- Calculate key performance indicators (KPIs)
- Perform city-wise and category-wise analysis
- Build a KPI dashboard for visualization
- Interpret results for business decision-making

---

## 📁 Dataset Details
- Total Records: 20
- Cities: Delhi, Mumbai, Bangalore, Chennai, Hyderabad, Pune, Jaipur, Kolkata
- Attributes:
  - Order ID
  - Customer Name
  - Restaurant Name
  - Food Category
  - Order Amount
  - Delivery Time
  - Distance
  - Rating
  - Status

---

## 🛠 Tools Used
- Microsoft Excel

---

## 📈 Key Analysis Performed
- Total Revenue Calculation using `SUM`
- Average Delivery Time using `AVERAGE`
- Highest & Lowest values using `MAX` and `MIN`
- Delivery Success Rate using `COUNTIF`
- Category-wise Rating using `AVERAGEIF`
- Order Ranking using `RANK`
- Revenue by Category using `SUMIF`

---

## 📊 Key Insights
- Average delivery time: ~34 minutes
- Delivery success rate: 85%
- Biryani has the highest average rating
- Delhi and Mumbai generate highest revenue
- High delivery time leads to cancellations

---

## 📉 KPI Dashboard
A dashboard was created in Excel to summarize:
- Total Orders
- Total Revenue
- Average Delivery Time
- Average Rating
- Fastest Delivery
- Highest Order Value

---

## ⚠️ Limitations
- Small dataset (20 records)
- No real-time data
- No machine learning used
- External factors like traffic/weather not included

---

## 🏁 Conclusion
This project shows how Excel can be used as a powerful tool for data analysis and decision-making in real-world scenarios like food delivery platforms.
